Objetivo:

El objetivo de este trabajo es desarrollar un análisis específico de la tendencia turística en el sector hotelero en España principalmente en los meses de verano.

Fuentes:

Frontur: http://estadisticas.tourspain.es/es-es/estadisticas/frontur/microdatos/paginas/default.aspx
INE: http://www.ine.es/prodyser/micro_frontur.htm

Archivos:

PFM.Rmd > Archivo del código usado para generar el informe.
PFM.html > Informe del proyecto.
Importación y limpieza de datos > Carpeta que contiene los datos que faltaban de Frontur para 2015, ya que estos solo llegaban hasta 					  Septiembre, el resto se extrajeron del INE. Este código se usó para crear los csvs Ft13, Ft14 y Ft15.

Provincias > Shapefiles de las Provincias Españolas.

Ft13.csv > Datos limpiados del año 2013. Son los que se uan en PFM.Rmd
Ft14.csv > Datos limpiados del año 2014. Son los que se uan en PFM.Rmd
Ft15.csv > Datos limpiados del año 2015. Son los que se uan en PFM.Rmd

Frontur2013.txt > Datos brutos extraidos de las fuentes
Frontur2014.txt > Datos brutos extraidos de las fuentes
Frontur2015.txt > Datos brutos extraidos de las fuentes
